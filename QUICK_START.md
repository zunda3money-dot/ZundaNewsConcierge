## ⏱ 3 ステップ (合計 7 分)

### 🔹 Step 1 - Gemini の API キーを取る (3 分)

> **これは何?**: 「AI に記事を要約してもらう権利」。完全無料です。

#### ⚠️ 英語画面が出ても焦らないで

AI Studio は英語のサイトです。**Chrome をお使いなら**、画面の何もない部分を**右クリック → 「日本語に翻訳」**を選ぶと自動で日本語化されます。

#### 手順

1. [https://aistudio.google.com/apikey](https://aistudio.google.com/apikey) を開く (API キー画面に直接ジャンプ)
2. Google アカウントでサインイン (普段使っている Gmail のアカウントで OK)
3. **画面右上または中央の青いボタン「Create API key」** をクリック
   - もしダイアログが出たら「**Create API key in new project**」を選ぶ (新しいプロジェクトを作る方が後で管理しやすいです)
4. `AIzaSy` で始まる**長い英数字**が表示されます → **コピーボタン**を押す
5. メモ帳 (Windows なら「メモ帳」アプリ) を開いて貼り付け、いったん保存

#### ⚠️ 大事なルール

- このキーは**パスワードのようなもの**。SNS に貼ったり人に見せたりしない
- メモ帳に貼って閉じる前に保存 (USB メモリや LINE のキープに残すと安心)

#### ✅ 完了サイン
メモ帳に `AIzaSy` から始まる長い文字列がある状態。

---

### 🔹 Step 2 - Discord Webhook を作る (1 分)

> **これは何?**: 「ニュースの配達先住所」。Discord の通知チャンネルが受け口になります。

#### 2-1. サーバーを準備する (持ってない人だけ)

Discord を開く → 左の縦並びアイコンの一番下の方にある **「＋」 (緑色の丸)** をクリック → 「**オリジナルの作成**」→「**自分と友達のため**」を選ぶ → 名前は何でもOK (例: `投資ニュース`)。30 秒で完成。

#### 2-2. 通知用チャンネルを開く

サーバーに最初から `#general` または `#一般` というチャンネルがあるはず。それを使って OK。
(専用チャンネルが欲しい人は、サーバー名の右の `+` から `朝のニュース` 等を新規作成しても OK)

#### 2-3. Webhook を作る

1. **チャンネル名 (例: `# 一般`) の右にある小さい歯車アイコン ⚙️** をクリック
   - 💡 歯車が見つからない場合: チャンネル名の上にマウスを乗せる/タップすると現れます
2. 左メニュー **「連携サービス」** → **「ウェブフック」** ボタン → **「新しいウェブフック」**
3. 「Spidey Bot」のような名前が自動で付きます (好きな名前に変えても OK、例: `投資秘書`)
4. **「ウェブフック URL をコピー」** ボタンを押す
5. メモ帳に貼り付け (Step 1 のキーの下に並べる)

#### ⚠️ 大事なルール

URL は**パスワード相当**。漏れると誰でもあなたのチャンネルに投稿できます。SNS スクショで写り込まないように注意。

#### ✅ 完了サイン
メモ帳に `https://discord.com/api/webhooks/...` で始まる URL が貼ってある。

---

### 🔹 Step 3 - リポジトリ複製 + Secret 登録 + テスト (3 分)

> **これは何?**: 自分専用のコピーを GitHub に作って、キーを安全に保管 → 動作確認、をまとめて。

#### 3-1. 自分のリポジトリを作る (30 秒)

1. このページの一番上に戻り、右上の **緑色の「Use this template」**ボタン → **「Create a new repository」**
   - ⚠️ **間違えやすい**: その左の青い「**Code**」ボタンではない、その右の**緑色のボタン**です
2. **Repository name** に好きな名前 (例: `my-news-concierge`)
3. **Private** にチェック ✅ (個人情報を世界に公開しないため)
4. 一番下の **「Create repository」** ボタン

#### 3-2. Secret 登録ページに直接ジャンプ (1 分)

ここが一番迷子になりやすいので、**直接リンクを使います**:

新しく作ったリポジトリ ( `my-news-concierge` 等 ) の URL の末尾に **`/settings/secrets/actions`** をつけたページを開く。
例: `https://github.com/<あなたのID>/my-news-concierge/settings/secrets/actions`

または:
- リポジトリ画面の上のタブから **Settings** → 左メニュー下の方 **「Secrets and variables」** → **「Actions」**

ここで **「New repository secret」** ボタンを押して、以下 **2 つ**を 1 つずつ追加:

| Name 欄 (**完全コピペ**してください) | Value 欄 (Step 1〜2 のメモから貼る) |
|---|---|
| `GEMINI_API_KEY` | `AIzaSy...` (Step 1) |
| `DISCORD_WEBHOOK_URL` | `https://discord.com/api/webhooks/...` (Step 2) |

⚠️ **大事**:
- Name は**手打ちしない**、必ず上の表からコピー貼り付け (大文字小文字・アンダースコアが完全一致しないと動きません)
- Value は前後にスペースが入っていないか確認

#### 3-3. Actions を有効化 (10 秒)

リポジトリの上タブから **Actions** をクリック → 緑色の **「I understand my workflows, go ahead and enable them」** ボタンを押す。

#### 3-4. テスト実行 (1 分)

1. **Actions** タブの左サイドバーから **「Test Setup (セットアップ確認)」** をクリック
2. 右上 **「Run workflow」** ドロップダウン → 緑色の **「Run workflow」** ボタン
3. 30 秒待つと新しい実行行が現れる → 緑のチェック ✅ になれば成功
4. **Discord アプリ**を開いて、選んだチャンネルに「✅ セットアップ確認: Discord Webhook OK」 のメッセージが届いていれば**完成!**

#### ❌ 赤い ✗ になったら

その行をクリック → 「test」ジョブを開く → **結果サマリー (📊 結果サマリー という表)** が**日本語**で「どこが NG か」教えてくれます。
よくあるのは: Secret の Name のスペル違い (`GEMINI_API_key` のように一部小文字になっている等)。

#### ✅ 完了サイン
Discord に確認メッセージが届いた!

## 📨 Gmail で受け取りたい人へ (+2〜4 分)

Discord ではなく**メール派**の人は、Step 2 を以下に差し替え。

⚠️ **前提条件**: Google の **2 段階認証が ON** であることが必要 (アプリパスワード機能の前提)。
2 段階認証は [https://myaccount.google.com/security](https://myaccount.google.com/security) で確認できます。OFF なら、まずそこを ON にしてください (電話番号登録 + SMS 認証で 3 分)。

<details>
<summary>📥 Gmail 受信のセットアップ (クリックで展開)</summary>

### Step 2-Gmail: アプリパスワードを発行

1. [https://myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords) を開く (アプリパスワード画面に直接ジャンプ)
   - 💡 「このページが見つかりません」と出たら、2 段階認証がまだ OFF です。先にそちらを ON に
2. アプリ名に `Zundamon News` と入れて **「作成」**
3. 16 桁の英数字 (例: `abcd efgh ijkl mnop`) が出る → **メモ帳にコピー**
   - 💡 **スペースは入ったままで OK**、ツール側で自動除去します
   - ⚠️ このページを閉じると 2 度と見られないので必ずメモ

### Step 3-Gmail: Secret を **4 つ**登録

| Name (完全コピペ) | Value |
|---|---|
| `GEMINI_API_KEY` | Step 1 の `AIzaSy...` |
| `GMAIL_FROM` | あなたの Gmail アドレス (例: `your-name@gmail.com`) |
| `GMAIL_APP_PASSWORD` | 上記 16 桁 (空白入りでも OK) |
| `EMAIL_TO` | 受信先メアド (自分宛なら GMAIL_FROM と同じ値) |

### Step 4-Gmail: Test Setup を実行

Gmail 受信トレイに「✅ Zundamon News - セットアップ確認 OK」 が届けば完成。
**迷惑メールフォルダ**も念のため確認してください (初回のみ振り分けられることがあります)。

</details>

**Discord と Gmail を併用したい人**: 両方の Secret を登録すれば**両方に届きます**。
