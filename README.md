## ⚠️ 無料枠運用の現実 (実機検証ベースで率直に)

このツールは「**Google アカウントだけで月 0 円運用**」が魅力ですが、それは **無料枠の制約を受け入れる前提** です。実際に 1 週間運用してみて分かったこと:

| 項目 | 実態 |
|---|---|
| 配信時刻のズレ | **設定 06:13 / 19:13 JST → 実際は約 3〜6 時間遅延** することがある (GitHub Actions 無料枠の cron はキュー優先度が低い) |
| 朝の配信 | **比較的安定** (アジア時間帯は Gemini が空いている) |
| 夜の配信 | **失敗しやすい** (遅延後の実行時刻が米国西海岸朝のピーク = Gemini が混む時間帯にぶつかる) |
| 失敗時の挙動 | エラーが Discord/Gmail に通知される (黙って止まらない) |
| 翌回への影響 | なし (1 回失敗しても、次の cron は通常通り発火) |

### 確実に届けたい人へ

- **そのまま無料運用**: たまに失敗・遅延するけど、ほとんどの日は届く。EP19 デモ用や個人の習慣として十分
- **Claude Haiku 課金 ($0.30〜$0.50/月)**: `config.yaml` の `ai_provider: claude` に変えて Anthropic API キーを足すだけで、Gemini ピーク時間帯の影響を受けなくなる
- **GitHub Pro 課金 ($4/月)**: cron 遅延が大幅に改善する (Actions の優先度が上がる)

両方足しても**月 700 円以下**で「**ほぼ確実に届く運用**」になります。
ただし**無料運用でも実用十分**なので、まずは無料で試してから判断するのがおすすめ。

## ❓ よくある質問

**Q. 自分のPCは起動しておかないとダメ?**
A. いいえ。GitHub Actions がクラウドで動くので、あなたのPCは常にオフでも毎朝届きます。

**Q. 投資アドバイスをしてくれる?**
A. いいえ。記事の要約と重要度判定だけです。投資判断は必ずご自分で。

**Q. 保有銘柄を他人に知られない?**
A. Template から作るリポジトリを **Private** にすれば、あなた以外は誰も見られません。SETUP_GUIDE の Quest 2-1 Step 3 で Private を選択してください。

**Q. 「保有スクショから AI が config 自動生成」はどこに画像をアップロードするの?**
A. **Google AI Studio** (Quest 1-2 で使った場所) のチャット画面に、ブラウザでドラッグ&ドロップします。ただし**無料の Gemini に渡した画像は Google の AI 学習に使われる可能性**があるため、**事前に口座残高や個人情報をマスクしてください**。詳細手順は [SETUP_GUIDE.md](./SETUP_GUIDE.md) の Quest 3-1 お手軽ルート参照。マスクが面倒な人は手動ルートを選択するか、Claude / ChatGPT の課金アカウントを使う (こちらは学習されません) のがおすすめです。

**Q. Gemini と Claude の違いは?**
A. Gemini は無料で速い。Claude Haiku は有料だが文章が洗練される傾向。`config.yaml` の `ai_provider` で切替可能。

**Q. RSS ソースを追加できる?**
A. はい。`sources.yaml` に URL を足すだけ。
