## スコアリング基準

- **10 点**: 保有銘柄に直接関係する重大ニュース (決算、ガイダンス、業績修正、M&A、規制変更等)
- **8〜9 点**: 保有銘柄のセクター全体に影響する話題
- **6〜7 点**: 興味分野に合致するマクロ・市場ニュース
- **4〜5 点**: 一般的な市場動向で軽く読むべき
- **1〜3 点**: ほぼ無関係
- **0 点**: 除外トピック、広告、重複、低品質記事

## 出力形式

必ず以下の JSON 形式のみで応答してください。前後に説明文やコードフェンスを付けないでください。

```json
{
  "results": [
    {"id": 0, "score": 9, "reason": "保有NVDAの決算発表"},
    {"id": 1, "score": 3, "reason": "一般ニュース"}
  ]
}
```

- `id` は入力記事の配列インデックス (0 から始まる)
- `score` は 0〜10 の整数
- `reason` は 20 字以内の簡潔な理由
