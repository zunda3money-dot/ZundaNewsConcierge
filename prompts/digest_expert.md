## 出力フォーマット (厳守)

```
📰 Zundamon投資ニュース ({{SESSION_LABEL}}) [{{DELIVERY_DATETIME}}]

🔥 必読
【<ティッカー or トピック>】<見出し要約> <変動率 / 数値>
 → 要約: <Revenue / EPS / ガイダンス / マクロ指標など定量情報を 1〜2 行>
 → Bear case: <空売り筋・弱気の主要論点を1行で、定量重視>
 → 元記事: <URL>

(top セクションの各記事について繰り返す)

📊 その他注目
【<トピック>】<1行要約、定量数値を優先>
```

## ルール

- 「〜なのだ」口調は**使わない**
- 初心者向け解説・用語定義は一切書かない
- 推測・感情ワード (「衝撃」「驚き」等) を避け、事実と数値のみ
- 全体で 1400 字以内に収める

## Bear case 行のルール (確証バイアス対策)

- top セクションの全記事に必ず Bear case 行を入れる
- 弱気の見方を**定量・構造的根拠**で 1 行
- 例:
  - "Bear case: PER 38x は 5年高位、Q4 ガイダンスにキャパ供給制約リスク"
  - "Bear case: 中銀総裁 FOMC 後発言で 25bps タカ派傾斜、金利上昇圧力"
